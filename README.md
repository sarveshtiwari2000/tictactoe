# tictactoe

To run using default configuration, execute: </br>

**gradle run** <br/>
<br/>
or <br/>
<br/>
**java PlayGame**

To change the configuration file settings:

**tictactoe.model.GameConfig**
    
## default values

playfield size 3x3 <br/>
first player : O <br/>
second player : X <br/>
computer player : R
